AndroidTask

abstract class AndroidTask(val taskList: AndroidTaskList<AndroidTask>)

Stores and retrieves VTODO iCalendar objects (represented as Tasks) to/from the OpenTasks provider.

Extend this class to process specific fields of the task.

The SEQUENCE field is stored in Tasks.SYNC_VERSION, so don't use Tasks.SYNC_VERSION for anything else.
