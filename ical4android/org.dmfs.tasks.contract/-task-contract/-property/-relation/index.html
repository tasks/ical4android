Relation

interface Relation : TaskContract.PropertyColumns

Relations of a task.

When writing a relation, exactly one of RELATED_ID or RELATED_UID must be present. The missing value and RELATED_CONTENT_URI will be populated automatically if possible.

PARENT_ID is updated automatically if possible.
