TRIGGER_TIMEZONE

const val TRIGGER_TIMEZONE: String

Purpose: This column/property specifies the timezone of the absolute trigger time. The corresponding datetime is stored in TRIGGER_TIME. The value of a timezone can be:

  1. the id of a Java timezone to represent the given timezone.

  2. null to represent floating time. If an invalid value is passed, the Timezone is ignored and interpreted as UTC. See [https://tools.ietf.org/html/rfc5545#section-3.8.2.4] Type: String
